### Description

This example generates a random graph with vtkRandomGraphSource and display it using a force-directed layout. The graph has 5 vertices and 4 edges with a fixed seed for reproducibility.

**vtkRandomGraphSource → vtkGraphLayoutView (force-directed layout)**

- [vtkGraphLayoutView](https://www.vtk.org/doc/nightly/html/classvtkGraphLayoutView.html) displays the graph with a force-directed layout.
- [vtkRandomGraphSource](https://www.vtk.org/doc/nightly/html/classvtkRandomGraphSource.html) generates a random graph with a specified number of vertices and edges.
