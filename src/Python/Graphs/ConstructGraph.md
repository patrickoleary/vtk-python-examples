### Description

This example constructs a simple undirected graph with two vertices and two parallel edges, then display it using a force-directed layout. Demonstrates the basic vtkMutableUndirectedGraph API for adding vertices and edges.

**vtkMutableUndirectedGraph → vtkGraphLayoutView (force-directed layout)**

- [vtkGraphLayoutView](https://www.vtk.org/doc/nightly/html/classvtkGraphLayoutView.html) displays the graph with a force-directed layout.
- [vtkMutableUndirectedGraph](https://www.vtk.org/doc/nightly/html/classvtkMutableUndirectedGraph.html) builds the undirected graph.
